# News Driven Sentiment Analyzer 

这是一个基于 Python 的全自动量化交易系统，结合了 **情感分析 (NLP)** 和 **机器学习** 来预测股价走势。

## 主要功能
- **数据采集**: 自动抓取新闻数据与股价历史.
- **情感分析**: 使用 NLP 技术分析新闻标题情绪.
- **模型训练**: 预测次日涨跌.
- **回测系统**: 模拟交易策略并评估收益.

## 如何运行
1. 安装依赖: `pip install -r requirements.txt` (如果有的话)
2. 运行主程序: `python -m src.main`
