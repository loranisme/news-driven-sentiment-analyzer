# News Driven Sentiment Analyzer

This is a fully automated quantitative trading system built with Python. It combines sentiment analysis (NLP) and machine learning to predict stock price direction.

## Key Features

- Data Collection: Automatically gathers news data and historical stock prices.
- Sentiment Analysis: Uses NLP techniques to evaluate the sentiment of news headlines.
- Model Training: Predicts whether the stock will rise or fall in the next trading session.
- Backtesting System: Simulates trading strategies and evaluates performance.

## How to Run

### Install Dependencies

Install the required Python packages for the project.

### Run the Main Program

```bash
python -m src.main
```
